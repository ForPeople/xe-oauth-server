  • 앱 관리

성능을 위해 비정상적으로 많은 API 요청만 제한되고 일상적인 API 요청은 별다른 제약 없이 사용할 수 있는 효과가 있습니다.
이 기능을 사용하기 위해서 Redis 연결이 필요합니다.

순간적으로 제한 요청 이상의 콜이 발생한다면 429 에러(Too Many Request)를 반환합니다.
Header의 Rate-Limiting-Meta 를 참조하시면 429 에러를 피할 수 있습니다.

사용자들이 앱 아이콘을 직접 올릴 수 있도록 설정할 수 있습니다.
